📊 Fed Interest Rate Tracker – Real-Time Automation with n8n + Google Sheets

This project automates the daily tracking of U.S. Federal Reserve interest rates by connecting the St. Louis Fed API to Google Sheets using n8n. The result is a live-updated spreadsheet with historical rate data, a dynamic chart, and email notifications — all without manual input.

🔧 Workflow Steps

  1. Scheduled Trigger: Runs once a day.
  2. HTTP Request: Pulls the latest interest rate from the St. Louis Fed API.
  3. Edit Fields: Extracts and formats the date and value.
  4. Google Sheets Append: Adds the data to the spreadsheet.
  5. Email Notification: Sends a summary update.

⚙️ Tools Used

  • n8n (self-hosted via ngrok)
  • St. Louis Fed API
  • Google Sheets API
  • Gmail Node for email alerts
